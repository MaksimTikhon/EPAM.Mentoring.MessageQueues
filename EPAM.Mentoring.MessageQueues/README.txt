1. Не обязательно устанавливать службы, достаточно просто запустить консольные приложения клиента и сервера сооответственно.
2. При запуске клиента создается папка ..\EPAM.Mentoring.MessageQueues\DocumentСaptureService\bin\Debug\in в нее можно добавлять изображения 
по 1му (потоковое сканирование), также можно создать папку самому и добавить файлы до запуска.
Пример изображений в папке Images (с корректным и некорректным форматами).
3. При запуске сервера создается папка ..\EPAM.Mentoring.MessageQueues\CentralServer\bin\Debug\out, куда сохраняются сформированные pdf.
4. Клиент отсылает серверу информацию о некоторых настройках, которые сохранятся в файл - ..\EPAM.Mentoring.MessageQueues\CentralServer\setting.csv
5. При изменении файла .\EPAM.Mentoring.MessageQueues\CentralServer\timeoгt.xml сервер сообщит клиенту о новом таймауте ожидания.
